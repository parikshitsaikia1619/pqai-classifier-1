# PQAI Classifier Service

REST API for accessing ML models trained for associating labels (e.g., CPC subclasses) with
text snippets.

## Routes

|      | Method   | Endpoint                   | Comments                                         |
| ---- | -------- | -------------------------- | ------------------------------------------------ |
|      | `GET`    | `/classify`                | Return subclass predictions for given text       |
